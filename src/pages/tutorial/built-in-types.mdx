## Core values

Sapphire includes the everyday value types you expect:

| Type | Examples |
| --- | --- |
| `Int` | `0`, `42`, `-7` |
| `Float` | `3.14`, `-0.5` |
| `Bool` | `true`, `false` |
| `String` | `"hello"` |
| `List` | `[1, 2, 3]` |
| `Map` | `{ name: "Ada", score: 95 }` |
| `Nil` | `nil` |
