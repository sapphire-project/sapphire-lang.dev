## Gradual typing

Sapphire is gradually typed: you can add `name: Type` and `-> Type` where they help, and lean on inference elsewhere. The type checker enforces the annotations you choose; untyped code still runs. Below, `clamp` is fully annotated, then `abs` and `factorial` show more patterns (recursion, guards, early return).
