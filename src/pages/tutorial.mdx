## Gradual typing

Sapphire is gradually typed: you can add `name: Type` and `-> Type` where they help, and lean on inference elsewhere. The type checker enforces the annotations you choose; untyped code still runs. Below, `clamp` is fully annotated, then `abs` and `factorial` show more patterns (recursion, guards, early return).

<section id="imports">

## Imports and local projects

The browser runner executes one file at a time, but local Sapphire programs can split code across files with relative imports:

```sapphire
import "./geometry/point"
import "../shared/helpers"
```

Top-level classes and functions from imported files become available to the importing file.

</section>
